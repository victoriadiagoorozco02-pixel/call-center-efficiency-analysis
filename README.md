# Call Center Performance Analysis - CallMeMaybe

## Project Description
Identifying inefficiencies in call center operations to improve service quality and reduce operational costs through data analysis.

The analysis combines exploratory data analysis, statistical testing, and dashboard visualization to generate actionable insights that enhance service performance.

---

## Objective
Identify inefficient operators by analyzing key metrics such as missed calls, waiting time, and outbound activity, validating findings through statistical testing.

---

## Dataset
The dataset includes information about:
- Incoming and outgoing calls  
- Waiting times  
- Missed calls  
- Operator activity  

---

## Tools Used
- Python (Pandas, NumPy)  
- SciPy (statistical testing)  
- Matplotlib / Seaborn  
- Tableau  

---

## Analysis Process

1. Data cleaning and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. KPI definition  
4. Identification of inefficient operators  
5. Hypothesis testing  
6. Dashboard development in Tableau  

---

## Key Insights

- No statistically significant differences were found in missed calls or waiting time between efficient and inefficient operators.  
- The number of outbound calls is the main factor associated with operator inefficiency.  
- Operational performance is more influenced by proactive call management than reactive call handling.  

---

## Recommendations

- Improve outbound call management through training and performance tracking  
- Define KPIs focused on proactive activity  
- Optimize workload distribution among operators  
- Use dashboards for continuous performance monitoring

## Conclusion

The analysis shows that operator efficiency is primarily driven by outbound call activity rather than inbound performance. These insights support data-driven decisions to improve productivity and service quality.

---

## Limitations

- Non-normal data distributions  
- Potential external factors not included in the analysis  

---

## 💼 Business Problem

Call centers often struggle to identify inefficient operators, leading to poor service quality and increased operational costs.
